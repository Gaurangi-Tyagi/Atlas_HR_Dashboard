Atlas HR Analytics Dashboard
Overview

This project presents an end-to-end HR Analytics Dashboard built using Power BI to analyze workforce data from the Atlas company.

The solution provides insights into employee distribution, performance, demographics, and attrition, enabling data-driven HR decision-making.

Objectives
Analyze employee distribution across departments and roles
Monitor employee performance and satisfaction
Identify key drivers of attrition
Provide actionable business recommendations

Tools & Technologies
Microsoft Power BI
Power Query (ETL)
DAX (Data Analysis Expressions)
CSV datasets

Key KPIs
Total Employees: 1470
Active Employees: 1233
Inactive Employees: 237
Attrition Rate: 16.12%

Key Insights
Attrition rate (~16%) indicates moderate employee turnover
Higher attrition observed among employees with low tenure
Frequent business travel is linked to increased attrition
Majority workforce lies in the 18–35 age group
Technology and Sales dominate employee distribution
Employee satisfaction trends help identify attrition risk

Business Recommendations
Improve onboarding to reduce early attrition
Address work-life balance issues for frequent travelers
Focus on high-risk departments and roles
Monitor satisfaction metrics to identify at-risk employees
Promote diversity and inclusive hiring practices
